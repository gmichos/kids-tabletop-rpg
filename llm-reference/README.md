# llm-reference/

Markdown source text for every deliverable, kept here purely as reference
material for AI-assisted editing of this project. These files are **not**
part of the published site — `index.html` and the pages under `content/`
never link here. The canonical, shipped format for every deliverable is the
matching `.html` file under `content/NN-name/`.

When editing content with an LLM, use these `.md` files as the easiest
plain-text source to read/modify, then hand-port the change into the
corresponding `content/NN-name/*.html` file (there is no build step that
generates one from the other — see the root `CLAUDE.md`).
